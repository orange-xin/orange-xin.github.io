蛇形矩阵-对角线型
发表于 2020-07-04| 更新于 2020-09-09|C语言

题目要求

  将自然数1、2、3、···、N*N逐个顺序插入方阵中适当的位置，这个过程中沿斜线进行，构成蛇形排列阵。如N=4时的蛇形排列阵为：

请编写C语言程序按以上规则输出N=5时的蛇形排列阵。

算法思路：

  观察上述矩阵的数值增长线路，如下图所示：

  • 初始状态：行 i = 0，j = 0 时，数值为1；
  • 当移动到下一个位置时，需要把当前设定的k值变成下一步要移动的方向状态；
  • k = 1时，下一步的移动方向可能是
    • 位于矩阵的第一行：k = 2；
    • 位于矩阵的最后一行：k = 4；
  • k = 2时，下一步的移动方向可能是
    • 位于矩阵的最后一行：k = 1；
    • 位于矩阵其他位置：k = 2；
    • 位于矩阵的第一列：k = 3；
  • k = 3时，下一步的移动方向可能是
    • 位于矩阵的最后一列：k = 2；
    • 位于矩阵的第一列：k = 4；
  • k = 4时，下一步的移动方向可能是
    • 位于矩阵的第一行：k = 1；
    • 位于矩阵的最后一列：k = 3；
    • 位于矩阵其它位置：k = 4；
